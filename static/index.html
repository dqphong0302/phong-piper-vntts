POST /tts

Text-to-Speech (hỗ trợ Vietnamese text preprocessing)

curl -X POST https://vitts.phongdang.io.vn/tts \
  -H "Content-Type: application/json" \
  -d '{"text":"Xin chào","voice":"banmai","speed":1.0,"pitch":1.0}' \
  -o speech.wav

POST /asr

Speech-to-Text (Whisper small, hỗ trợ 6 ngôn ngữ)

curl -X POST https://vitts.phongdang.io.vn/asr \
  -F "file=@audio.wav" \
  -F "language=vi"

POST /v1/audio/speech

OpenAI TTS compatible

curl -X POST https://vitts.phongdang.io.vn/v1/audio/speech \
  -H "Content-Type: application/json" \
  -d '{"input":"Xin chào","voice":"banmai","response_format":"mp3"}' \
  -o speech.mp3

POST /v1/audio/transcriptions

OpenAI Whisper compatible

curl -X POST https://vitts.phongdang.io.vn/v1/audio/transcriptions \
  -F "file=@audio.wav" -F "language=vi"

POST /asr/srt

Xuất phụ đề SRT với timestamps

curl -X POST https://vitts.phongdang.io.vn/asr/srt \
  -F "file=@audio.wav" -F "language=vi" -o transcript.srt

POST /tts/long

TTS cho văn bản dài (tự động chia chunk)

curl -X POST https://vitts.phongdang.io.vn/tts/long \
  -H "Content-Type: application/json" \
  -d '{"text":"Đoạn văn rất dài...","voice":"banmai"}' \
  -o speech.wav
